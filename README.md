# 🎮 Rock Paper Scissors Game

## 📌 Overview
A command-line Rock, Paper, Scissors game developed in Python where the player competes against the computer using randomly generated choices.

## ✨ Features
- Play against the computer
- Random computer moves
- Input validation
- Multiple rounds
- Easy-to-use command-line interface

## 🛠️ Technologies Used
- Python 3
- Random Module

## ▶️ How to Run

```bash
python game.py
```
